# 学术侦探系统 - 跨语种科研趋势发现系统

## 系统概述

"学术侦探"是一个多Agent协作的跨语种科研趋势发现系统，旨在通过智能化的数据采集、分析和可视化，帮助研究者发现学术前沿和蓝海研究机会。

## 核心功能

### 1. 多源数据采集模块
- **数据源**：支持arXiv、CNKI等学术平台
- **更新机制**：实现定期增量更新，确保数据时效性
- **语言支持**：同时支持中英文文献采集

### 2. 研究主题识别与分析
- **趋势分析**：运用NLP技术识别新兴研究方向
- **机会指数**：建立评估模型，识别"值得进入但竞争少"的蓝海领域
- **热点监测**：实时跟踪学术热点和前沿发展

### 3. 跨语言知识整合
- **概念对齐**：自动对齐中英文概念，如"大模型推理"↔"LLM Reasoning"
- **信息流动**：打破中英文研究者的"信息茧房"
- **知识图谱**：构建跨语言学术知识网络

### 4. 可视化与报告生成
- **交互式图谱**：展示研究主题间关联关系
- **自动报告**：生成包含蓝海方向的专业分析报告
- **趋势图表**：多维度可视化数据趋势

### 5. 系统反思与优化
- **月度复盘**：自动评估系统预测准确率
- **反馈循环**：持续优化预测模型和机会指数算法
- **性能监控**：实时追踪系统运行状态

## 系统架构

### Agent角色分工

#### 📊 Data Collection Agent
- **职责**：多源学术数据采集
- **功能**：
  - arXiv RSS源抓取
  - CNKI等中文学术平台数据采集
  - 数据清洗和标准化
  - 增量更新机制

#### 📈 Trend Analyst Agent
- **职责**：研究趋势分析和机会识别
- **功能**：
  - 主题提取和聚类
  - 趋势热度计算
  - 机会指数评估
  - 蓝海领域识别

#### 🌐 Cross-Language Agent
- **职责**：中英文概念对齐和跨语言整合
- **功能**：
  - 概念自动翻译
  - 语义相似度计算
  - 知识对齐映射
  - 跨语言信息桥接

#### 📊 Visualization Agent
- **职责**：数据可视化和报告生成
- **功能**：
  - 知识图谱构建
  - 交互式图表生成
  - 研究报告自动生成
  - 结果导出和分享

#### 🔄 Reflection Agent
- **职责**：系统反思和性能优化
- **功能**：
  - 预测准确率评估
  - 系统瓶颈分析
  - 性能指标监控
  - 优化建议生成

#### 🎯 Orchestrator Agent
- **职责**：协调各Agent协作和任务调度
- **功能**：
  - 工作流程规划
  - 任务分发协调
  - 错误处理恢复
  - 结果整合输出

## 技术栈

### 后端技术
- **框架**：LangChain + LangGraph
- **大模型**：Doubao-Seed（支持思考模式和视觉理解）
- **数据库**：PostgreSQL + SQLAlchemy
- **消息队列**：基于Python Queue的Agent通信机制
- **网络搜索**：Coze集成的联网搜索能力

### 前端技术
- **可视化**：Plotly + D3.js（交互式知识图谱）
- **图表**：Matplotlib + Seaborn（统计分析图表）
- **报告生成**：Jinja2模板 + HTML/CSS

### 数据处理
- **文本处理**：jieba中文分词 + NLTK英文处理
- **数据格式**：JSON + XML（学术文献数据）
- **实时更新**：RSS订阅 + 定时任务

## 使用指南

### 快速开始

1. **系统初始化**
   ```bash
   # 安装依赖
   pip install -r requirements.txt
   
   # 初始化数据库
   python scripts/init_database.py
   ```

2. **启动系统**
   ```bash
   # 启动主服务
   python -m src.main
   ```

3. **使用示例**
   ```python
   from src.agents.agent_simple import process_academic_detective_request
   
   # 分析AI研究趋势
   result = process_academic_detective_request("分析当前AI研究领域的趋势")
   ```

### API调用示例

```python
# 采集学术数据
result = process_academic_detective_request(
    "采集机器学习和深度学习相关的最新论文"
)

# 分析研究趋势
result = process_academic_detective_request(
    "分析人工智能领域的研究趋势和热点"
)

# 识别蓝海机会
result = process_academic_detective_request(
    "识别值得进入的蓝海研究方向"
)

# 跨语言概念对齐
result = process_academic_detective_request(
    "对齐中英文AI研究概念"
)

# 生成研究报告
result = process_academic_detective_request(
    "生成AI领域研究机会报告"
)
```

### 数据库配置

系统使用PostgreSQL作为主要数据存储，包含以下主要表结构：

- `papers`：论文信息表
- `research_topics`：研究主题表  
- `trend_analysis`：趋势分析表
- `opportunity_scores`：机会评分表
- `concept_mappings`：概念映射表
- `system_metrics`：系统指标表
- `task_logs`：任务日志表
